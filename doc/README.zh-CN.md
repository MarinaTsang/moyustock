### 简介

摸鱼看盘是一个 Chrome 扩展，在任意网页的右下角嵌入一个轻量浮窗，实时显示自选股票行情、分时走势和大盘指数，无需切换标签页。

专为需要在工作中隐蔽看盘的用户设计：没有异动时低调隐形，有情况时及时提醒。

---

### 功能介绍

#### 实时行情
- 支持 **A 股**（沪 `sh` / 深 `sz` / 北 `bj`）、**港股**（`hk`）、**美股**（`us`）
- 显示股票名称、当前价格、涨跌幅
- 本地采样生成分时走势线（SVG）
- A 股涨跌停检测（±9.8%），特殊标签标注
- 标题栏常驻上证指数

#### 自选与轮播
- 最多同时显示 3 只股票，超过 3 只时每 5 秒循环轮播
- 在扩展弹窗中管理自选，支持格式：`600519`、`TSLA`、`00700`
- 轮播位置跨页面刷新持久化

#### 三态存在感
| 状态 | 外观 | 触发条件 |
|------|------|---------|
| **静默** | 透明度 35%，极低调 | 非交易时段 |
| **活跃** | 透明度 85%，正常显示 | 交易中，无显著波动 |
| **关键** | 透明度 100%，彩色边框 | 任一股票涨跌 ±3% 持续 10 秒以上 |

关键态区分涨跌方向（金色边框=上涨，蓝色边框=下跌），并显示针对性情绪提示。

#### Stealth 伪装模式
将浮窗切换为**新闻模式**——隐藏所有股票数据，改为显示近期财经新闻，看起来像一个新闻阅读器，有效规避旁观者注意。点击浮窗内切换按钮即可切换。

#### 盘前/盘后摘要
摘要弹窗在关键时间节点自动弹出：
- **08:45（盘前）**：美股隔夜行情、财经日历
- **15:05（盘后）**：A 股行业热力图、机构资金流向、自选股复盘

也可在弹窗中手动刷新查看。

#### AI 个股摘要
当自选股涨跌幅超过 **5%** 时，自动从 Google News 抓取近期相关新闻，生成一段简洁的原因摘要。同一股票 30 分钟内冷却，不重复触发。

#### 主题跟随系统
自动跟随浏览器/系统的深色/浅色设置：
- **深色模式**：深灰背景，金色（涨）/ 钢蓝（跌）
- **浅色模式**：白色背景，深琥珀（涨）/ 钢蓝（跌）
- 两种模式均避免醒目的红绿配色，保持低调

#### 快捷键
| 快捷键 | 功能 |
|--------|------|
| `Alt + Q` | 显示/隐藏浮窗（老板键） |
| `Alt + Shift + D` | 调试面板开关 |

#### 隐私说明
- 无需账号，无需登录
- 所有行情数据直接从公开接口获取
- 不上传任何用户数据到第三方服务器
- 自选股列表仅存储在本地 `chrome.storage.local`

### 安装方法

1. 克隆或下载本仓库
2. 打开 Chrome，进入 `chrome://extensions`
3. 开启右上角的**开发者模式**
4. 点击**加载已解压的扩展程序**，选择项目文件夹
5. 在任意网页按 `Alt + Q` 唤起浮窗，或点击扩展图标

### 文件结构

```
├── manifest.json        # 扩展清单（MV3）
├── background.js        # Service Worker：行情代理、定时摘要
├── content.js           # 注入每个网页：浮窗完整逻辑
├── content.css          # 浮窗样式（CSS 变量支持深/浅色主题）
├── popup.html           # 扩展弹窗界面
├── popup.js             # 弹窗逻辑
├── shared.js            # 公共工具（行情解析、代码规范化）
├── market.js            # 交易时段、交易所判断
├── trend.js             # 分时走势线渲染（SVG）
├── widgetView.js        # 浮窗 DOM 构建
├── panelView.js         # 设置面板视图
├── carouselView.js      # 轮播/切换视图
├── anomalyMonitor.js    # 异常波动检测模块
└── crawler.js           # 数据爬取（板块、资金流）
```
